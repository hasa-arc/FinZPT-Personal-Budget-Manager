# FinZPT-Personal-Budget-Manager

FinZPT — Personal Budget Manager

FinZPT is a command-line personal finance tracker built with Python. It allows users to record income and expenses, organize spending by category, and view a financial summary at any time.

Features:

- Add multiple income sources with amounts
- Track expenses by category (food, health, rent, etc.)
- View a full financial summary with total income, total expenses, and remaining balance
- Data is saved automatically and loaded on next use

Built with: 
Python — using core concepts including lists, dictionaries, loops, functions, file handling, and JSON storage.
